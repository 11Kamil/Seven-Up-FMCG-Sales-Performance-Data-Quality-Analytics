# Portfolio Project Summary

## Project title

FMCG Sales Performance & Data Quality Analytics

## Summary

Built an end-to-end FMCG sales analytics pipeline using Power Query, MySQL, and Power BI. The project cleaned CRM and SAP-style sales data, validated customer and product records, reconciled order values, measured sales and delivery performance, analyzed target-vs-actual gaps, identified declining customers, and flagged payment collection risk.

## Business value

The project helps a sales management team understand not only revenue performance, but also the reliability of the data behind the report. It surfaces operational issues such as unmapped customers, invalid product codes, order-value mismatches, delivery gaps, target-matching gaps, and outstanding customer balances.

## Key results

- Total sales: ₦600.34M
- Total cases: 190,925
- Orders: 3,400
- Active customers: 676
- Average order value: ₦176,570.15
- Invalid CRM orders: 47
- Invalid product-code lines: 66
- Order-value exceptions: 75
- Strict 3-month declining customers: 104
- Targets without actual sales: 1,587
- Actual sales without targets: 349
- Outstanding amount: ₦157.51M
- Collection rate: 64.0%

## Tools

Power Query, MySQL Workbench, Power BI, Excel, AI-assisted documentation and debugging.

## What makes it strong

The project compares two workflows—Power Query and SQL—and validates that both produce the same core results. This demonstrates analytical thinking, data-quality control, SQL capability, BI reporting, and business interpretation.
