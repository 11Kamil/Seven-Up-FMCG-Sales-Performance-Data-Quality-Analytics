# Screenshots to Review Before Uploading

Project process screenshots were not automatically included in the public-ready repository because screenshots can expose personal information such as browser tabs, email addresses, local file paths, or account names.

Before adding screenshots to GitHub or Kaggle:

1. Crop out browser tabs and personal details.
2. Remove email addresses and local account names.
3. Keep only the relevant tool area: Power Query, MySQL Workbench, Power BI, or dashboard canvas.
4. Use clear filenames such as:
   - `mysql_raw_import_row_counts.png`
   - `mysql_invalid_crm_orders.png`
   - `mysql_target_vs_actual_validation.png`
   - `powerbi_sales_overview.png`
5. Place sanitized screenshots in this folder.

For LinkedIn, use final dashboard screenshots rather than raw process screenshots.
