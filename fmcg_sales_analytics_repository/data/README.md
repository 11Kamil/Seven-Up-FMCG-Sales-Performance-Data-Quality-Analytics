# Data

This folder contains the synthetic raw CSV data used for the FMCG Sales Analytics project.

The data simulates CRM and SAP-style sales operations tables for a beverage/FMCG company. It is intended for learning and portfolio demonstration.

## Files

- `01_crm_customers.csv`
- `02_crm_sales_rep_route_master.csv`
- `03_sap_product_master.csv`
- `04_sap_price_list.csv`
- `05_sap_sales_orders_header.csv`
- `06_sap_sales_order_lines.csv`
- `07_sap_delivery_invoice.csv`
- `08_sap_customer_payments.csv`
- `09_sales_targets.csv`
- `10_data_dictionary.csv`

## Note

Do not use real company data in a public repository unless it has been anonymized and approved for sharing.
